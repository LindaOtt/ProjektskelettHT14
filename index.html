Vad är Sass?

Sass är en preprocessor. Den gör det möjligt att använda variabler, inkapslade regler, mixins och nedärvning.

.sass och .scss

Det finns två syntax som används för Sass. Den senaste är känd som SCSS (Sassy CSS) och är en förlängning av CSS.

Sass äldre syntax är ett mer komprimerat sätt att skriva CSS. Den använder bl a indentering istället för parenteser för att nästla selektorer. Filer med denna äldre syntax har filändelsen .sass istället för .scss.

Hur används Sass?

Sass kan användas som:

  1. Ett verktyg från kommandoraden
  2. En fristående Ruby-modul
  3. En plugin

Hur används Sass?

Gör så här:

  1. Windows: installera Ruby från http://rubyinstaller.org/
  2. Skriv ”gem install sass” i konsolen
  3. Skriv ”sass input.scss output.css” för att köra Sass. Scss-koden processas om till css-kod, som sparas i output.css.
  4. ”sass –watch input.scss:output.css” låter Sass kolla efter förändringar och uppdatera CSS-filen automatiskt.

Variabler

I Sass-variabler kan man lagra färger, font stacks, eller vilket annat CSS-värde man vill återanvända.

För att skapa en variabel använder man $-symbolen.

Variabler

SASS-koden processas och tar variablerna vi definierat för $font-stack och $base-color och skapar normal CSS-kod med värdet för våra variabler.

Exempel (SCSS):

$font-stack: Verdana, sans-serif;
$base-color: red;
body {
 font: $font-stack
 color: $base-color
}

Resultat:

body {
 font: Verdana, sans-serif;
 color: red;
}

Nesting

I SASS kan man ”nästla” CSS-selektorerna så att de följer samma hierarki som i HTML.

Exempel (SCSS):
nav {
 ul {
  margin: 0;
  padding: 0;
  list-style: none;
 }
 li { display: inline-block }
}

Nesting

Resultat:
nav ul {
 margin: 0;
 padding: 0;
 list-style: none;
}
nav li {
 display: inline-block;
}

Mixins

En mixin är en grupp av CSS-deklarationer som en vill återanvända. En kan även skicka in värden för att göra sin mixin mer flexibel.

Ett bra område är prefix, som -webkit- och -moz-.

Mixins

För att skapa en mixin använder jag @mixin och ger det ett namn.

Variabeln $radius används också inom parenteserna för att lägga in en radie.

När man vill använda sin mixin, börjar man med @include följt av namnet på sin mixin.

Ex (SCSS):
@mixin border-radius($radius) {
 -webkit-border-radius: $radius;
 -moz-border-radius: $radius;
 -ms-border-radius: $radius;
 border-radius:
}
.box { @include border-radius(10px); }

Mixins

Resultat (genererad CSS):
.box {
 -webkit-border-radius: 10px;
 -moz-border-radius: 10px;
 -ms-border-radius: 10px;
 border-radius: 10px;
}

Operatorer

I Sass kan man använda operatorerna +, -, *, / och %

Det kan exempelvis komma till nytta när man kodar responsiva layouter och vill räkna ut vad ett elements vidd är i procent av dess föräldraelement.

Ex (SCSS):
.container {
 width: 100%;
}
 
article {
 width: 600px / 960px * 100%;
}

Operatorer

Resultat (genererad CSS):
.container {
 width: 100%;
}
 
article {
 width: 62.5%;
}
